# Voice calibration guide

Use this when the user provides a writing sample, brand voice, or strong style preference.

## What to match

Match rhythm: sentence length, paragraph length, pace, and how often the writer uses fragments.

Match vocabulary: simple or technical, plain or polished, direct or literary.

Match posture: warm, restrained, bold, skeptical, founder-like, expert-like, friendly, institutional, or editorial.

Match openings and closings: how the voice enters a topic and how it exits without sounding like an assistant.

Match restraint: some brands should be vivid; some should be almost invisible.

## What not to copy blindly

Do not copy typos unless the user wants exact preservation.

Do not preserve unclear structure if the task is to improve clarity.

Do not imitate quirks that hurt trust, such as fake certainty, vague claims, or unsupported proof.

Do not use one house persona for every client. Uniform voice is the failure this skill prevents.

## No sample provided

Default to direct, concrete, and grounded. Use opinion only where the format supports it. Prefer specifics, mechanics, tradeoffs, and proof over mood.

## Quick calibration questions

Ask only when the piece is voice-sensitive and the missing answer would change the rewrite.

Useful questions:

- Can you share one paragraph that sounds like you or the brand?
- Should this feel founder-led, institutional, editorial, or sales-driven?
- Are there words or tones the brand never uses?
- Should the copy feel more restrained or more energetic?

If the user wants immediate output, proceed with the grounded default and use placeholders where proof is missing.
